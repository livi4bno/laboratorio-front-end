Tag

O elemento é usado para diminuir o ênfase visual de um trecho do texto, normalmente deixando-o com um tamanho de fonte menor em relação ao texto ao redor

Historicamente, o era usado apenas para mostrar o texto em fonte menor. Mas com o tempo, o significado foi ampliado para indicar informações secundárias ou menos importantes, como notas de rodapé, avisos legais, copyringt, etc.

Exemplo de uso:

Este é um parágrafo normal. Este é um texto menor.
